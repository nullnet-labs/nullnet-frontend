# Project Nullnet SSR Frontend
Next.JS front-end / SSR application layer for the fullstack Web project [Nullnet](https://github.com/nullnet-labs).

This is a [Next.js](https://nextjs.org) project bootstrapped with [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

To run it locally, have Node.js installed and run the development server:

```
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You should see something similar to this:
